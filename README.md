# Ansible Role: marvel-nccr.editors

Installs a selection of text editors on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.editors`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.editors
